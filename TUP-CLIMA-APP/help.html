Ayuda Clima app v.1.0

Esta app te permite consultar el clima de una ciudad en particular utilizando la API de
OpenWeatherMap.

Para poder consultar el clima, primero deberá Agregar una ciudad y luego seleccionar
la Ciudad en cuestión desde la página Clima.
